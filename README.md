
# Expo-Kryptom

Native code bridge for Expo using Kryptom to add cryptographic capabilities


## Installation

Install expo-kryptom

### npm

```bash
  npm install @icure/expo-kryptom
  npx expo prebuild
```

### yarn

```bash
yarn add @icure/expo-kryptom
npx expo prebuild
```
